Terrain Image Compression - Github

Although a unique use of DNNs, it is interesting. This project trained a deep neural net on recunstructing an image with as few neurons as possible, in an attempt to compress the image. This is my first (interesting) project with DNNs. It's a few years old now, and I'm curious to see what I would do different now.

Objective

Given a variety of terrain images, use a small neural network to "store" the image in as little data as possible.
