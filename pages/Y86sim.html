Y86 Simulator - Github

Example Input & Output

Y86 Pipelined CPU simulator. The simulator is written to follow the following architecture diagram, and I have also included the Y86 instruction set.

Objective

I worked on this project with Brandon Davis, a peer and friend, in order to better understand collaboration through github, machine code, assembly, writing our own assembler, and CPU architecture as a whole. After completion, I had a better grasp on common architecture concepts such as data forwarding, bubbling, stalling, and the importance of memory read and write concepts.
